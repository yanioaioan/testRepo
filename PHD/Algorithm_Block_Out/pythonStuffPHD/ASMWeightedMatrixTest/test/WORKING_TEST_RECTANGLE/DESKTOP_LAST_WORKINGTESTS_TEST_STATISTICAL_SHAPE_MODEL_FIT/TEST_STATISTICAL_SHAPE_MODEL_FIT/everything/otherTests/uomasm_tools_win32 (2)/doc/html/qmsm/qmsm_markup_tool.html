qmsm_markup_tool: Tool for creating points and curves on an image

A tool which allows a user to
  • Load in an image
  • Annotate it with points and connecting curves
  • Save the points and curves to .pts and .crvs files
This is useful for creating the first template for a shape model.
Use the mouse wheel to zoom in and out.
Once the points have been saved, they can be editted using the qmsm_shape_editor

Menu Options

File Menu

  • Open Image
  • Load Points
  • Load Curves
  • Save Points
  • Save Curves
  • Graph Props - Display panel allowing size/colours of points/curves to be tweaked
  • Unzoom - Show whole image

Help Menu

  • Help: Display summary of functionality

Interaction Modes

Drag Mode Click and drag to move viewport around
Add Point Mode Click to add new point
Add Curve Mode Click new position to extend point from currently highlighted point
When in "Add Curve" mode, you can link to an existing point by clicking on it.
To start a new curve, click on the currently selected point (to deselect it), then click in a new position to create a new point and select it. A second click elsewhere with create a new point and connect to it.

Undo/Redo

The undo/redo buttons allow you to do just that.
